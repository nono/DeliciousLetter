# DeliciousLetter
=================

Simple library to send email with delicious' links of the week.

## Requirements
 * ruby 1.9.3
 * delicious account
 * gmail account to send email by smtp


## Installation
  * config: `cp config/development.yml.example config/development.yml`
  * gem: `bundle install`

## How it works
It's simple: `./bin/deliciousletter`
